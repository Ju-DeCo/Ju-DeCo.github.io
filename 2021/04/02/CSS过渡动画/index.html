CSS过渡动画

CSS Transition 呈现的是一种过渡效果，过渡可与变形同时使用，transition 属性的初始值根据 4 个子属性的默认值而定：

(1) transition-property ：执行变换的属性

(2) transition-duration ：变换的延续时间

(3) transition-timing-function ：在延续时间段，变换的速率变化

(4) transition-delay ：变换的延迟时间

定义过渡属性

transition-property 属性用来定义动画的 CSS 属性名称，即指定在哪个属性发生变化时执行过渡效果，主要有这几个值：none（没有属性改变）、all（所有属性改变）和元素属性名，元素属性名可以指定元素的某个属性值。下例使用 transition-property 属性指定过渡效果的属性为 background-color，此时看不到过渡效果，因为没指定过渡时间。

定义过渡时间

transition-duration 属性用来指定元素过渡效果的持续时间，单位为秒，默认过渡时间为 0 秒。在例1的 transition-property 属性的后面增加 transition-duration 属性即可看到过渡效果：

在鼠标经过元素上时，颜色会慢慢过渡到 gold ，但宽度没有过渡效果，如果要实现宽度的过渡效果，需要修改 transition-property 属性为 all 或 “background-color,width”。

定义延迟时间

transition-delay 属性用来指定动画开始前的等待时间，单位为秒过毫秒，默认值为 0 。

定义过渡效果

transition-timing-function 属性用来指定过渡效果，有如下取值：

(1) ease ：过渡效果逐渐变慢，为默认值

(2) linear ：匀速过渡效果

(3) ease-in ：加速过渡效果

(4) ease-out ：减速过渡效果

(5) ease-in-out ：过渡效果首先加速，然后减速

(6) cubic-bezier ：允许自定义一条时间曲线
