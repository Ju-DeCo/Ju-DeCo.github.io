CSS关键帧动画

过渡动画只能定义第一帧和最后一帧两个关键帧，而关键帧动画能定义任意多的关键帧。关键帧动画使用 @keyframes 来定义，一般 0% (可用 from 代替) 和 100% (可用 to 代替)这两个关键帧是必须定义的，例如

animation-name

animation-name 用来定义动画的名称，该名称就是使用 @keyframes 关键字创建的动画名。若要使用多个动画，该属性后面的值要用逗号分隔。

animation-duration

animation-duration 属性用来定义动画播放的持续时间，单位为秒，默认值为 0 。

animation-timing-function

animation-timing-function 属性和过渡动画的 transition-timing-function 属性相同，用来控制动画的播放方式，并且接收的值也相同。

animation-delay

animation-delay 属性用来定义动画的延迟播放时间，单位为秒，默认值为 0 。

animation-iteration-count

animation-iteration-count 属性用来指定动画播放的循环次数，默认值为 1，infinite 表示无限次。

animation-direction

animation-direction 属性用来指定动画播放的方向，可取值如下：

(1) normal ：表示动画按正常顺序播放，默认值

(2) reverse ：表示动画反向播放

(3) alternate ：动画在奇数次正向播放，偶数次反向播放

(4) alternate-reverse ：动画在奇数次反向播放，在偶数次正向播放

animation-play-state

animation-play-state 属性可以用来暂停播放动画，该属性有两个值：running(默认) 和 paused(暂停正在播放的动画)。如果设置从 paused 改为 running 时，动画就会从暂停处继续播放。

animation-fill-mode

animation-fill-mode 属性用来控制当动画不播放时(播放完或有延迟未开始播放)，要应用到元素的样式。默认值为 none，还有如下取值：

(1) forwards ：产生动画的元素会保持最后一个关键帧属性，通常为from 或 to 关键帧

(2) backwards ：正向播放使用 from 关键帧定义的属性值，反向播放使用 to 关键帧定义的属性值

(3) both ：相当于 backwards 和 forwards 的组合，意味着在动画等待和动画结束状态，元素将分别应用动画第一帧和最后一帧的样式
