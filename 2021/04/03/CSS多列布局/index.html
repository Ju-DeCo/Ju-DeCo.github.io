CSS多列布局

属性 描述
column-count 设置元素被分隔的列数
column-width 设置列宽
column-fill 设置如何填充列
column-gap 设置列之间的间隔
column-rule 设置列边框的样式
column-span 设置元素应该横跨的列数
columns column-width 和 column-count 属性的简写形式

column-width 和 column-count

column-width 属性用于给列定义最小宽度，默认值为 auto，表示将根据 column-count 属性指定的列数自动动计算列宽，column-coount 属性用来指定文本显示的列数。通常将这两个属性简写成 columns。

column-gap

column-gap 属性用来指定列间距，该属性的默认值为 normal，相当于 1 em，如果 column-gap 和 column-width 加起来大于总宽度，就无法显示 column-count 指定的列数。

column-rule

column-rule 用来设置列边框，列边框不占用任何空间，设置 column-rule 属性不会导致列宽变化。它包含 3 个子属性：column-rule-width、column-rule-style 和 column-rule-color。边框宽度通常小于 column-gap 属性的值，否则可能会导致边框覆盖部分文字。

column-span

column-span 属性用来设置跨列标题，该属性有两个取值：默认值 none(表示不跨列) 和 all(表示跨越所有列)

column-fill

column-fill 属性用于统一列高，默认值为 auto，各列的高度随内容自动调整，设置为 balance 时，所有列的高度都设为最高的列高。
