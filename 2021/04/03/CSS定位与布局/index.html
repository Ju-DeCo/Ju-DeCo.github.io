CSS定位与布局

默认情况下，页面中使用正常流或静态流进行布局，在正常流中，页面中的块级元素从顶部向底部流动，行内元素从左向右流动。如果希望元素的内容出现在与正常流中不同的位置，有两个属性可以帮助实现：position 和 float。

position 属性

position 属性有如下取值：

(1) static ：与正常流相同，为默认值

(2) relative ：盒子的位置可以相对其在正常流中的位置出现偏移

(3) absolute ：将盒子对象从正常流中拖出

(4) fixed ：固定在屏幕的某个位置，不随用户滚动窗口而改变位置

盒子偏移属性

当盒子的 position 属性值为 relative、absolute 或 fixed 时，会同时使用盒子偏移属性指定盒子应如何定位，盒子偏移属性含义如下：

<1> top ：指定元素的顶部边缘

<2> left ：指定元素的左边缘

<3> bottom ：对于绝对定位(相对)属性，设置单位高于/低于包含它(正常位置)的元素底边

<4> right ：指定元素的右边缘

每个属性的取值可以是长度值、百分比或 auto。

z-index 属性

z-index 用来表示元素在叠加顺序上的立体关系。使用绝对定位和相对元素的元素常常会与其他元素出现重叠，这种情况下，默认的行为是第一个元素位于后来元素之下，z-index 属性用来指定重叠部分的上下层关系。该属性的值是数字，数字越大的显示在页面的上层。

float 属性

float 属性能够将某个元素从正常流中抽取出来，包含元素中其他元素内容会围绕关联有 float 属性的元素进行流动。当为元素指定 float 属性时必须设置 width 属性以指定盒子应该占据的宽度，否则元素将自动占据包含盒子的 100%的宽度。该属性取值如下：

(1) none ：默认值，盒子不发生浮动

(2) left ：盒子浮动到包含元素的左侧

(3) right ：盒子浮动到包含元素的右侧

(4) inherit ：从父元素继承 float 属性的值

clear 属性

clear 属性定义元素哪边不允许出现浮动元素，该属性可能取值如下：

(1) left ：左侧不能出现任何内容

(2) right ：右侧不能出现任何内容

(3) both ：左右两侧不能出现任何内容

(4) none ：默认值，允许两侧出现浮动内容
