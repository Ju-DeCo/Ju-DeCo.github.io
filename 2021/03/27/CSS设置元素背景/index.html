CSS 设置元素背景

属性 描述
background-color 指定背景颜色
background-image 指定一张作为背景的图片
background-position 指定背景图片的位置
background-size 指定背景图片的大小
background-origin 指定背景图片的定位区域
background-repeat 指定背景图片是否应该重复显示
background-clip 指定背景的绘制区域
background-attachment 指定背景图片应该固定于页面中的某个位置，以及在用户向下滚动页面时是否留在原地

background-color 属性

background-color 属性能够为任何元素指定单一实体色背景，取值与 color 属性值一样，可以有多种模式。

background-image 属性

background-image 属性用于为元素设置背景图像，元素的背景占据元素的全部尺寸，包括内边距和边框，但不包括外边框。默认情况背景图像位于元素的左上角，并在水平和垂直方向上重复。

如果同时使用 background-image 属性与 background-color 属性，则 background-image 属性拥有优先权。

background-position 属性

background-position 属性用来设置背景图像 (background-image 属性定义) 的起始位置，background-position属性的可取值如下：

值 描述
top left
top center
top right
center left
center center
center right
bottom left
bottom center
bottom right		 如果只规定了一个关键词，那么第二值是 “center”
x% y% 第一个值是水平位置，第二个是垂直位置，左上角是 0% 0%(默认值)，如果只规定了一个值，另一个值是 50%
xpos ypos 左上角是 0 0，单位是像素或任何其他 CSS 单位，如果只规定一个值，另一个值是 50%

background-size 属性

background-size 属性用来控制背景图像的尺寸，该属性的取值如下：

(1) length ：设置背景图片的高度和宽度，包括两个值。第一个值设置宽度，第二个值设置高度，如果只设置一个值，另一个值会被设置为 auto

(2) percentage ：以父元素的百分比来设置背景图像的宽度和高度，包括两个百分比值。第一个值设置宽度，第二个值设置高度，如果只设置一个值，另一个值会被设置为 auto

(3) cover ：使背景图像完全覆盖背景区域

(4) contain ：使背景图像完全适应内容区域

background-origin 属性

background-origin 属性规定 background-position 属性相对什么位置来定位。如果背景图像的 background-attachment 属性为 fixed，该属性不生效。该属性的取值如下：

(1) padding-box ：指定背景图像相对于内边距来定位

(2) border-box ：指定背景图像相对于边框来定位

(3) content-box ：指定背景图像相对于内容来定位

background-repeat 属性

background-repeat 属性规定图像的平铺模式，默认情况背景图像在水平和垂直方向上重复，背景图像的位置是根据 background-position 属性设置的。该属性可取值如下：

(1) repeat ：默认，背景图像在水平和垂直方向上重复

(2) repeat-x ：背景图像在水平方向上重复

(3) repeat-y ：背景图像在垂直方向上重复

(4) no-repeat ：背景图像仅显示一次

background-clip 属性

background-clip 属性用来规定背景的绘制区域，取值与 background-origin 属性相同，表示背景图像被剪裁到相应的边框处。

background-attachment 属性

background-attachment 属性用于设置背景图像是否固定或随着页面的其余部分滚动，该属性取值如下：

(1) scroll ：默认值，背景图像会随着页面其余部分的滚动而移动

(2) fixed ：当页面的其余部分滚动时，背景图像不会移动
