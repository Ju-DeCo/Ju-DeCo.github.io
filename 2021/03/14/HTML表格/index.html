创建表格

创建基本表格

HTML表格由4个标签创建，分别是表格标签<table>、行标签<tr>、表头标签<th>、和表格数据标签<td>。 表格中每一行都包含在一个 <tr> 元素中。

表格边框

可以使用 border 属性设置表格的边框宽度。默认情况下不指定该属性，表格边框为0，浏览器不显示表格边框。

border 属性的值表示边框的宽度，单位为像素。

表格的结构标签

为在源代码中清楚的区分表格的结构，HTML提供了<thead>、<tbody>、<tfoot>三个标签，分别对应表格的表头、表体和表尾。还可用 CSS 为这三个标签添加不同的样式风格。

如果表格太长导致在屏幕中显示不全，那么表头和表尾可以始终保持可见，而为表体设置滚动条。
