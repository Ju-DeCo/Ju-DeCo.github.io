基本链接

创建链接使用 <a> 元素，应包含两部分：

  • <a> 元素的 href 属性，即链接的目标，可以是某个网址或文件的路径

  • 链接文本或图像，单击文本或图像将跳转到 href 属性指定的链接地址。

例子中给出的是百度的 URL 地址，URL(Uniform Resource Locator,统一资源定位符)是互联网上的标准地址，互联网上的每个文件都有唯一的URL，URL包含3个关键组成部分：

  • 协议(scheme)：指明文件将以何种方式传输。大多数Web页面使用 HTTP(HyperText Transfer Protocol,超文本传输协议)传输信息，因此大多数网页的地址以 “http://” 开头，而 “https://” 是 HTTP 的一种更安全形式
  • 主机地址(host address)：通常是网站的域名，如 baidu.com 。我们看到的以 “www” 为前缀实际并非域名的一部分。主机地址还可以是数字形式的IP地址
  • 文件路径(file path)：以一个正斜线(/)开头，然后以一个文件名结束，也可以没有文件名

绝对地址和相对地址

  • 绝对地址：指向目标路径的完整描述，一般指向本站点外的文件或URL

  • 相对地址：相对于当前页面的路径，一般指向本站点内的文件

    有如下几种类型的相对地址：

    1. 同一目录：可以直接使用文件名进行引用

    2. 子目录：若在TV中建一个页面，Music是其子目录，则可应用如下方式指向index.html链接

    3. 父目录：从目录中创建指向父目录的链接，可以使用 “../” 又称”上一级“符号。

    4. 从根目录出发：以“/”开头，表示根目录

<base>标签

当浏览器遇到相对地址时，会将URL地址转换为完整的绝对URL地址。<base>标签可以为页面指定基础地址，该标签没有关闭标签。基础地址的值在<base>标签的 href 属性中指定。

链接到电子邮件地址

此链接是一种特殊的超链接，单击该链接，会启动电子邮件程序并打开一个新邮件。

方法是在<a>元素中设置 href 属性,开头以关键字 mailto 开始，然后是冒号，最后是电子邮件的地址。

​ 以下是可用的电子邮件链接特性，方法是在邮件地址后附加一个问号并在之后以“名称/值”形式指定特性。

特性 描述
subject 为电子邮件添加主题行
body 向电子邮件主体中添加默认的消息内容
cc 向该地址抄送电子邮件。如果需要向多个电子邮件地址抄送，特性间使用 & 符号分隔
bcc 指定秘密抄送的电子邮件地址，秘密抄送的收件人之间互不可见。如果需要向多个电子邮件地址抄送，特性间使用 & 符号分隔

对个特性之间用”&”分隔，例如以下，默认主题为“读者来信”，抄送给”123456@qq.com“ 和 “654321@qq.com“

如果需要在主题中使用空格，不能直接使用空格而应该使用转义字符“%20”。

如果需要在主体中换行，应该使用“%0D%0A”。

创建锚点链接

锚点链接也叫书签链接，用于跳转到页面特定段落。

锚点链接创建有两步：

(1) 创建命名锚点

使用<a>标签的 id 或 name 属性。锚点名称区分大小写。

(2) 链接到命名锚点

创建锚点后，使用<a>标签的 href 属性链接，href 属性需要与对应的锚点的 id 属性值相同，并且在值的前面加上“#”。

除了可链接到当前网页中的锚点，还可以链接到其他页面的锚点，此时 href 属性值的格式为： 文件路径+文件名 #锚点名称。
