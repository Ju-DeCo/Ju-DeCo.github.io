CSS基本选择器和属性选择器

基本选择器

标签选择器

标签选择器直接将 HTML 标签作为选择器，如果要将相同的规则应用于多个标签，则可用逗号分隔。

一种特殊的标签选择器是通用选择器，用一个 (*) 表示，表示匹配文档中全部标签。

类选择器

类选择器能够将规则与一个或多个包含 class 属性的元素相匹配，class 属性的值匹配类选择器中指定的值。

在英文句号前添加限定标签只对该元素匹配的 class 生效。

如果对多个类编写相同的样式规则，可用逗号操作符按顺序排列类名即可。

ID选择器

ID 选择器只作用于 HTML 元素的 id 属性，并使用“#”进行标识。

CSS 3 属性选择器

CSS 3 遵循通用编码规则，选用 ^、$ 和 * 这三个通用匹配运算符。

​ (1) [abc^=”def”]：选择 abc 属性值以 “def” 开头的所有元素

​ (2) [abc$=”def”]：选择 abc 属性值以 “def” 结尾的所有元素

​ (3) [abc*=”def”]：选择 abc 属性中包含子串 “def” 的所有元素
