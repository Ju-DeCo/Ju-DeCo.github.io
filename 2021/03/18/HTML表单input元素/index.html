HTML表单–input元素

在表单中，用户输入数据时使用的文本框、复选框等都是通过<input>元素创建的，该元素必须在<form>元素中使用，用来创建表单的输入元素，如果不在<form>中使用，那么输入元素不会被<form>元素收集并传给服务器。<input>元素是空的，所以一般不需要闭标签，通常直接在开标签的后面加一个斜线。

<input>标签的 type 属性用来指定输入控件的类型，type 属性的值如下：

输入类型 type 类型 功能说明
文本框 text 定义单行输入字段，用于在表单中输入字母、数字等内容，默认宽度为 20 个字符
单选按钮 radio 定义单选按钮
复选框 checkbox 定义复选框
密码框 password 定义密码字段，输入的内容会以 "*" 或点的形式出现
提交按钮 submit 定义"提交"按钮，用于将表单数据发送到服务器
可单击按钮 button 定义普通可单击按钮，多用于通过 JavaScript 启动脚本
图像按钮 image 定义图像形式的提交按钮
隐藏字段 hidden 定义隐藏的输入字段
重置按钮 reset 定义重置按钮，可通过单击该按钮来清除表单中的所有数据
文件域 file 定义输入字段和浏览按钮，用于上传文件
e-mail 地址 e-mail 用于输入单个电子邮件地址或电子邮件地址列表，多个电子邮件地址可以使用逗号分隔的列表形式输入
颜色选择框 color 用于使用色盘选择颜色
日期 date 用于输入日历日期
日期和时间 datetime 用于输入国际标准时间的日期和时间
日期和时间 datetime-local 用于输入本地日期和时间
年和月 month 用于输入年份与月份
时间 time 用于输入由时、分、秒以及小数秒组成的时间
星期 week 用于输入有年份与周数构成的日期，例如 2021-W03，2021年第三周
数字输入框 number 用于输入数字
数字滑块 range 该控件通常以拖动的形式呈现，使用户可以从一个数字范围内选择取值
搜索文本框 search 用于输入搜索关键词
文本框 tel 用于输入电话号码
网站地址 url 用于输入网站的 URL

文本框和密码框

当 type 属性值为 text 或 password 时，通常还会用到<input>元素的如下几个属性：

(1) name ：几乎所有的<input>元素要设置该属性，该属性的值就是向服务器发送的"名称/值"对中的名称部分

(2) value ：该属性用来设置文本框提供的初始值，在表单加载后能看到该值

(3) size ：设置文本框的宽度，以字符为单位，该属性不影响用户可输入的字符数量，如 size 属性的值为20，用户任然可以输入 40 个字符

(4) maxlength ：该属性用来指定用户在文本框内可以输入的最大字符数，输入达到最大后，再输入也不会有新的字符加入

(5) placeholder ：用来为文本框设置输入提示信息，当文本框处于未输入状态且未获得光标焦点时，模糊显示输入提示文字

(6) required ：该属性是布尔属性，如果元素是空的，则不允许提交，同时浏览器提示用户必须输入内容

autocomplete 属性和 datalist 元素

autocomplete 属性可以允许网页作者控制是否缓存表单条目，将 autocomplete 设置为 off，则表示不保存表单值。

如果 autocomplete 属性为 on，则可以显式的指定候补输入的数据列表，此时需要用到 datalist 元素和 list 属性。

datalist 元素用于给输入框提供一个可选列表，该列表由 datalist 元素中的 option 元素创建，为了把 datalist 元素提供的列表绑定到某个输入框，需要使用 input 的 list 属性来引用 datalist 元素的 id。

复选框和单选按钮

checkbox 属性创建的是复选框控件，radio 属性创建的是单选框控件。这两个控件都有开和关两种状态，并可以通过设置 checked 属性，指明页面加载后的初始状态，区别在于：

(1) 当一组单选按钮共享同一名称时，只有其中之一可以被选定

(2) 不应将单选按钮作为指定开或关的单一表单控件使用，因为当单一单选按钮被选定后，就无法通过界面操作再次失去选定；而复选框可以单独出现

(3) 复选框也可以共享同一名称，作为复选框组出现，同一组中可以有多个复选框同时被选定

按钮

(1) submit ：创建单击时提交表单的按钮

(2) reset ：创建重置表单的按钮，点击后设置为页面载入时初始值的按钮

(3) button ：创建用于触发客户端脚本的按钮

(4) image ：创建带图片的提交按钮，单击后也会提交表单数据，通常需要把 src 属性和 alt 属性一起配合使用，还可以用 height 和 width属性指定图片的高和宽

对于按钮控件需要注意以下几点：

(1) 提交按钮和重置按钮都有默认显示的文本，也可通过 value 设置新的文本；而 type=”button”创建的按钮没有默认显示文本，必须使用 value 属性指定。

(2) 如果提供了 name 属性，那么 value 属性的值会作为表单控件的"名称/值"对的一部分发送到服务器

(3) 对于图片按钮，如果提供了 name 属性，那么单击后，发送的"名称/值"对中的"值"是用户单击图片按钮时的 x 及 y 坐标值对

(4) 图片按钮的 alt 属性为图片提供替换文本，在图片无法找到是将显示该信息

隐藏字段

若需要在页面间传递信息而不希望被用户看到，这就要用到隐藏字段，隐藏字段的创建只需将 type 属性设置为 hidden，然后指定 name 和 value 属性即可，提交表单时，name 和 value 属性将与表单其他元素一起发送至服务器。

文件上传控件

文件上传控件用于需要用户上传文件到服务器的情况，设值 type 属性为 file，在页面显示效果中会出现“浏览”按钮或“选择文件”按钮。使用该控件时，<form>元素的 method 属性值必须是 post，另外还需增加属性 enctype，值为 “multipart/form-data”，否则无法实现文件上传。对于<input>元素，还可以使用 multiple 属性设置允许一次性上传多个文件，使用 accept 属性指定可被选择上传的文件的 MIME 类型。
