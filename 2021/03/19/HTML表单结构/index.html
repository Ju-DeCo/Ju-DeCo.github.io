HTML表单结构

​ <fieldset>和<legend>两个元素可以将相关表单控件组织到一起

  • <fieldset>元素在表单控件组的四周添加边框
  • <legend>元素用来为<fieldset>元素指定标题

效果如下

<label>元素为控件创建标签，对于大多数表单控件，必须为其提供标签。使用表格布局表单不算规范，应该使用<label>元素。

disabled 与 readonly 控件

  • readonly 属性用来防止用户更改表单控件的值，但仍然可以通过脚本更改，任何该控件的名称与值仍然将被发送至服务器

  • disabled 属性会禁用表单控件或<fieldset>元素中的表单控件组，从而使用户无法更改，但可使用脚本重新激活控件，若不激活，名称与值就不会被发送到服务器

    readonly 属性在需要阻止访问者修改表单的某些数据时很有用，disabled 属性在用户完成某些操作前阻止用户与特定控件交互时用。以下表单控件不支持 readonly 属性：

    元素 元素 元素
    <input type=”checkbox”> <input type=”button”> <keygen>
    <input type=”radio”> <input type=”color”> <option>
    <input type=”submit”> <input type=”range”> <optgroup>
    <input type=”reset”> <button> <fieldset>
    <input type=”file”> <select>
