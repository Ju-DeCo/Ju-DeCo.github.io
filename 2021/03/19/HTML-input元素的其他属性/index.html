HTML input元素的其他属性

1. form 属性

​ 使用 form 属性，从属于表单的元素可以放在页面的任何地方。

2. formaction 属性

​ 该属性可指定单击提交按钮时要将表单数据发送到的地址。

3. formmethod 属性

​ 该属性可为每个提交按钮指定不同的提交方法。

4. formenctype 属性

​ 该属性用于提交按钮以指定表单在发送到服务器之前应该如何对表单内容进行编码。

5. formtarget 属性

​ 可对多个提交按钮分别使用 formtarget 属性来指定提交后在何处打开需要加载的页面。

6. autofocus 属性

​ 该属性为布尔属性，使用该属性后，当页面打开时，这些控件将自动获得光标焦点。一个页面上只能有一个控件具有该属性。
