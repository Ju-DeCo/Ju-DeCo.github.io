HTML下拉列表

创建下拉列表

下拉列表包含于 <select> 元素内，下拉列表中的每一个选项是一个<option>元素，位于<option>开闭标签之间的文本会作为选项的标签显示给用户。提交表单时所选选项的 value 属性值将作为 <select>元素的值传送到服务器。<option>元素的 selected 属性可设置一个默认选中的选项。

创建滚动选择框

<select>元素的 size 属性，可以创建一种滚动选择框，size 属性的值是希望可见的选项数量。

可选择多个选项的选择框

multiple 属性是一个布尔属性，使用该属性后，选择框会自动以滚动条选择框的形式出现。 若要进行多选，按住 Ctrl 键，然后单击要选择的选项即可。
