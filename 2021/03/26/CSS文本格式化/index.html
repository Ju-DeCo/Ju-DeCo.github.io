CSS文本格式化

Updated2021-03-28|CSS
|Post View:

CSS 文本格式化

属性 描述
color 指定文本颜色
text-align 指定文本水平对齐
vertical-align 指定文本垂直对齐
text-decoration 指定文本是否具有下划线、上划线或中划线
text-indent 指定从左侧边框起文本的缩进
text-transform 指定元素内容应全部为大写、小写或首字母大写
text-shadow 指定文本应具有投影
letter-spacing 控制字符间宽度
word-spacing 控制单词间距离
text-overflow 指定文本溢出包含元素时是否显示省略号
word-wrap 允许对长的、不可分割的单词或 URL 强制换行
white-space 指定空格是否应该被压缩、保留或阻止换行
direction 指定文本行文方向

color 属性

十六进制颜色代码

将 3 个介于 00 和 FF 之间的十六进制数链接起来设置颜色值，若十六进制的 3 组数各自成对，则可简写为 3 位。

颜色关键字

(1) 颜色名称 ：直接使用颜色的英文单词，例如 red、bule、orange 等

(2) transparent ：用来表示文本的颜色纯透明，可近似认为是 rgba(0,0,0,0)

(3) currentColor ：指当前的文字颜色

HSL 模式

HSL 模式通过色调(H)、饱和度(S)、亮度(L)三个颜色通道的变化以及它们的相互叠加来得到各式各样的颜色，使用方法如下：

各参数含义如下：

(1)h 表示色调(hue)，色调可为任意整数，0 (以及 360 或 -360)表示红色，60 表示黄色，120 表示绿色，180 表示青色，240 表示蓝色，300 表示洋红，当 h 值大于 360 时，实际的值等于该值摸 360 后的值

(2) s 表示饱和度(saturation)，是指颜色的深浅度和鲜艳程度。取值范围为 0 ~ 100%，其中 0 表示灰度（没有颜色），100% 表示饱和度最高（颜色最鲜艳）

(3) l 表示亮度(lightness)，取值范围为 0 ~ 100%，其中 0 表示最暗（黑色），100% 表示最亮（白色）

text-align 属性

text-align 属性会将文本在包含元素或浏览器窗口中对齐，该属性的取值有：left(左对齐)、right(右对齐)、center(居中)、和 justify(两端对齐)。

vertical-align 属性

vertical-align 属性用来设置元素的垂直对齐方式，该属性定义行内元素的基线相对于该元素所在行的基线垂直对齐。该属性的取值如下：

值 描述
baseline 与父元素的基线对齐（默认）
sub 将元素作为下角标
super 将元素作为上角标
top 将文本顶端及图片顶端与行内最高元素的顶端对齐
text-top 将文本顶端及图片顶端与行内最高文本的顶端对齐
middle 将元素垂直中点与父元素垂直中点对齐
bottom 将文本底部及图片底部与行内最低元素的底部对齐
text-buttom 将文本底部及图片底部与行内最低文本的底部对齐

除了上表中的取值，该属性的值还可以是 inherit、长度值或百分比值。inherit 表示从父元素继承 vertical-align 属性的值，也可使用行高属性（line-height）的百分比值来排列元素，长度值和百分比值可以是负值。

text-decoration 属性

text-decoration 属性用来添加文本修饰，取值如下：

值 作用
none 默认，定义标准的文本
underline 在内容下方添加一条线
overline 在内容顶部之上添加一条线
line-through 添加一条从中间穿过内容的线，通常用于指定标记为删除的文本

text-indent 属性

text-indent 属性用来为块级元素设置首行缩进量，对行内标记无效。该属性的值还可以使用负值，表示向前凸出相应的大小或字符。

text-shadow 属性

text-shadow 属性用来创建投影，该属性的值共有 4 个参数：

各参数作用如下：

(1) x-offset ：水平阴影，表示阴影的水平偏移距离，单位可为 px、em 或百分比等，值为正表示向右偏移，为负向左偏移

(2) y-offset ：垂直阴影，值为正向下偏移，为负向上偏移

(3) blur ：可选参数，表示阴影的模糊程度，单位可为 px、em 或百分比等。该参数的值不能为负，值越大阴影越模糊

(4) color ：可选参数，表示阴影的颜色

text-transform 属性

text-transform 属性用来指定元素内容的大小写形式，主要针对英文字母，可能的取值：none、captialize(大写每个单词的首字母)、uppercase(全部大写)、lowercase(全部小写)。

letter-spacing 和 word-spacing 属性

letter-spacing 属性用来设置字符间的距离（字距），word-spacing 属性用来设置单词间的距离，该属性对中文没有影响，而 letter-spacing 可以影响中文的文字间距。

white-space 属性

white-space 属性可控制空格，可能取值如下：

值 作用
normal 遵循正常的空格压缩
pre 像<pre>那样保留空格
nowrap 只有显式使用<br>元素指定时才进行换行
pre-wrap 保留空白符序列，正常进行换行
pre-line 合并空白序列，保留换行符

text-overflow 属性

文本溢出属性 text-overfolw 用来设置是否使用省略标记(…)标识对象内文本的溢出，该属性取值有两个：ellipsis(显示省略标记) 和 clip(不显示省略标记，而是将溢出的部分剪裁掉)。

direction 属性

direction 属性与 dir 属性类似，用于指定文本应该流动的方向，该属性的取值有：ltr(左向右)、rtl(右向左)、和 inherit(与父元素相同)。
