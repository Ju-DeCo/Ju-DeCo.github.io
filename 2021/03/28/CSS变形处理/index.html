CSS变形处理

CSS 增加了 transform 属性，可以实现文字或图像的变形操作，主要包括旋转、倾斜、缩放和移动。该属性接收一个或多个由空格分隔的应用于元素的变形函数。

旋转

旋转通过 rotate() 函数来实现，在旋转之前可使用 transform-origin 属性定义旋转的基点，rotate() 函数的参数是旋转角度，可以是正值（顺时针旋转）或负值（逆时针旋转）。

在三维变形中，还有三个旋转函数：rotateX()、rotateY()、rotateZ(),可让元素在任何轴上旋转。也可使用 rotate3d() 函数同时指定沿 3 个维度旋转，使用方法如下：

倾斜

倾斜可通过 skew() 函数来实现。skew() 函数可以带一个或两个参数，只有一个参数会使元素在 X 轴倾斜，第二个参数控制在 Y 轴上倾斜。skew() 可接收负值和较大的值，skew(90deg) 会让元素消失，大于90deg（或小于-90deg）的值会产生镜像效果，倾斜没有3d函数。

缩放

对元素的缩放可通过 scale() 函数来实现，该函数接收无参数的数字，scale(1) 是元素的默认尺寸，小于 1 的值使元素变小，大于 1 使元素变大。如果参数值为负值，则进行反转。

也可指定两个参数 scale(x,y)，使元素在水平和垂直方向上同时缩放，对于元素的 3D 缩放，有 scaleZ(sz) 和 scale3d(sx,sy,sz)两个函数，这两个函数单独使用时没有任何效果，需要配合其他变形函数一起使用。
