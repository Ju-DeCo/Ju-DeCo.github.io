CSS边框与边距

盒子模型

盒子模型的重要属性如下：

属性 描述
border 边框，用来将一个盒子的边界与周围其他盒子相分离，即使不可见，每个盒子都有边框
margin 从盒子边框到下一个盒子的距离(外边距)
padding 元素内容与盒子边距的距离(内边距)

border 属性

border 属性用来指定代表某一元素的盒子的边框应如何呈现，有如下 3 个可修改的属性：

(1) border-color ：指定边框的颜色

(2) border-style ：指定边框应为实线、虚线还是双股线等

(3) border-width ：指定边框具有的宽度

border-width 属性

border-width 属性用来设置边框的宽度，通常以像素为单位，也可使用任何绝对或相对单位，但不能使用百分比。还可使用：thin、medium 和 thick。使用以下属性分别指定盒子边框的底部、右侧、顶部和左侧宽度：border-bottom-width、border-right-width、border-top-width 和 border-left-width。

border-style 属性

border-style 属性用来指定边框的线性样式，可取值如下：

值 描述
none 默认值，不存在边框
solid 边框是一条实心线
dotted 边框是一系列的点
dashed 边框是一系列短线
double 边框是两条实心线
groove 边框具有切入效果
ridge 边框效果与groove相反
inset 边框使盒子看起来内嵌于页面中
outset 边框使盒子看起来突出于画布外
hidden 与 none 相同，但用于作为表格元素的边框冲突解决方案

也可使用以下属性分别指定盒子边框的底部、右侧、顶部和左侧的样式：border-bottom-style、border-right-style、border-top-style 和 border-left-style。

border-color 属性

border-color 属性能够改变盒子周围边框的颜色，取值与 color 属性相同，还可为每侧边框指定不同颜色：border-bottom-color、border-right-color、border-top-color 和 border-left-color。

使用缩略形式表达边框属性

与 font、background 等属性一样，可使用 border 属性同时指定边框的宽度、颜色和样式，顺序为 border-width、border-style 和 border-color，如果不指定某个值可以省略。

还可为每侧边框指定不同的宽度、颜色和样式：border-bottom、border-right、border-top 和 border-left。

padding 属性

padding 属性用来指定元素内容与边框之间的距离，也叫作内边距，该属性的值可取：像素、长度单位、百分比或关键字 inherit。

如果使用百分比，则以包含盒子的百分比计算，如果指定为 10%，则每一边取盒子的 5% 作为内边距。

元素的内边距默认不会继承，只有 padding 属性值为 inherit 的元素才会继承父元素的内边距。

与边框一样，也可分别指定每侧的内边距大小：padding-bottom、padding-top、padding-right、padding-left。

margin 属性

margin 属性用来控制盒子之间的空间，取值可是长度、百分比或关键字 inherit，也可分别指定每侧的外边距大小：margin-bottom、margin-top、margin-right、margin-left。

border-radius 属性

border-radius 用于将边框设置为圆角矩形，该属性可分别设置 4 个角的效果半径，分别对应：border-top-left-radius(左上角)、border-top-right-radius(右上角)、border-bottom-right-radius(右下角) 和 border-bottom-left-radius(左下角)。使用该属性如果省略值时，将采用对角线原则，不同数目的值对角产生的影响如下：

(1) 一个值 ：所有四个角都有相同的半径

(2) 两个值 ：第一个值时左上角和右下角，第二值是右上角和左下角

(3) 三个值 ：第一个是左上角，第二个值是右上角和左下角，第三个值是右下角

(4) 四个值 ：依次是左上角、右上角、右下角和左下角

border-radius 属性的值可以用 em、像素和百分比作为单位。也可设置椭圆角效果，即给出两组长度值：一组针对 X 轴，另一组针对 Y 轴，水平和垂直方向的值用斜线(/)分隔。

border-image 属性

border-image 属性用来为边框添加背景图片，边框图像可由单幅图像创建，这幅图像可在元素周边的边框中沿着不同的轴向进行裁剪或拉伸，图像被 4 条线分成 9 个切片，4个角切片用于创建元素边框的 4 个角，剩下 4 个边切片由 border-image 属性用来填充元素边框的 4 条边。border-image 属性基本语法如下：

source 是准备用来作边框图像的文件 URL 地址

slice 由 4 个值决定，决定了切片的尺寸，值可以是百分比或数字，也可像 border-radius 一样省略一些值

repeat 是指边框进行平铺的方式，可使用 round、stretch、space 和 repeat 中的两个值

​ (1) round ：图像会被平铺，直至铺满整个区域，如果平铺后切片的数量与区域不匹配，就会对图像进行相应的缩放

​ (2) stretch ：图像会被拉伸，直至填满整个区域

​ (3) repeat ：图像会被平铺，直至填满整个区域

​ (4) space ：图像会被平铺，直至填满整个区域。如果平铺后切片的数目与区域不匹配，就会调整图像之间的间距以填满整个区域

box-shadow 属性

box-shadow 属性用法和text-shadow 属性相似，可以使用一个或多个投影，用法如下：

(1) h-shadow ：指定阴影的水平偏移量

(2) v-shadow ：指定阴影的垂直偏移量

(3) blur ：可选参数，只能为正值，标志阴影模糊效果，值越大越模糊

(4) spread ：可选参数，如果为正值，那么整个阴影将延展扩大，为负值，则阴影缩小

(5) color ：可选参数，指定阴影颜色

(6) inset ：可选参数，表示将外部阴影改为内部阴影
