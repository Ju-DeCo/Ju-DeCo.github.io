在 HTML 中使用 CSS

CSS简介

CSS 的中文全称是层叠样式表（Cascading Style Sheets）,是一种用来表现 HTML 或 XML 等文件样式的编程语言。CSS 不仅可静态修饰网页，还可以配合各种脚本语言动态对网页中元素进行格式化。

内联样式

​ 如果只是简单的对某个元素单独定义样式，可以使用内联样式。

style 属性的值是若干个 CSS 属性/值对。CSS 规则要求属性名后面紧跟一个冒号，然后是属性值，每一个样式规则以分号结束，最后一个样式规则的结尾可以不加分号。

定义内部样式表

CSS 样式是通过 <style>标签嵌入页面中的，<style>元素位于<head>元素内，用于在网页内包含样式规则。在使用<style>元素时，尽管并非强制，但应该永远带有 type 属性，且属性值为 “text/css”。

链接外部样式表

将共同的样式抽取出来，保存为一个独立的 CSS 文件，通过在文档的 <head> 部分使用<link>元素，引用外部样式表。<link>元素用于在网页中描述两个文档间关系，并且该元素永远是空元素，当与样式表一同使用时，必须带有两个属性：rel 和 href 。rel 属性用于指定包含链接的文档与链接指向文档间的关系，值为“stylesheet”；href 属性用于指定链接指向的文档的 URL。

导入外部样式表

导入外部链接样式表的语法是@import，然后是关键字 url 和要导入的外部样式表的 URL，最后以分号结束，@import 指令必须在<style>标签中使用，而且必须在样式表中所有其他类型的规则之前使用。

@import 和 link 的区别在于：

(1) <link>是 XHTML 标签，除了加载 CSS 外，还可以定义 RSS 等其他事务，@import 只能加载 CSS

(2) 链接引用 CSS 时，在页面载入时同时加载；@import 需要页面完全载入后才加载

(3) <link>无兼容问题，@import 在低版本的浏览器不支持

(4) <link>标签支持使用 JavaScript 控制 DOM 以改变样式，而 @import 不支持

CSS继承

大多数元素可以从父元素那里继承样式属性，但是有些样式属性（例如边框）不可以继承，对于可以继承样式规则的属性，也可以重新定义，覆盖继承的值。在使用 CSS 继承时，需要重点关注 font-size 属性，虽然也继承，但对于不同标签，继承的效果不同。

！important 指示符

如果某个特定的规则不希望被其他规则覆盖，那么可以使用！important 指示符。

当使用！important 指示符时，要确认放在样式规则的最后，否则会被忽略。
