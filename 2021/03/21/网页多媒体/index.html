向网页中添加图片、音频及视频

使用<img>元素

<img>元素是一个独立标签，该元素必须包含两个元素：src 和 alt 。src 属性指定图片的来源，值是个 URL；alt 属性是对图片进行描述的文本，图片无法显示就用该文本代替。除此之外，还常用带 width 和 height 属性，用来指定图片的宽和高，值通常以像素为单位，也可以是百分比。

使用图片作为链接

将文本置于<a>开闭标签之间即可将图片作为链接

使用图像映射

除了一幅图像整体作为超链接外，还可以将一副图像分成几个部分，分别链接到不同页面，此时使用图像映射来实现。创建图像映射需要用到 <map> 和 <area>两个元素：

  • <map>元素用于定义客户端图像映射。图像映射指带有可单击区域的一幅图像，<img>中的 usemap 属性可引用<map>中的 id 或 name 属性，所以创建图像映射时，应同时向<map>添加 id 和 name 属性
  • <area>元素用于定义图像映射中的区域，当用户单击这个区域时即可链接到指定的页面。<area>元素永远嵌套在<map>内部，通过 shape 属性指定所选区域的形状，通过 coords 属性指定区域的坐标

area 元素的 shape 属性可取值如下：

(1) rect 或 rectangle ：创建矩形图像区域

(2) circ 或 circle ：创建圆形图像区域

(3) poly 或 polygon ：创建多边形图像区域

相应的 coords 属性为图像区域的坐标，图像左上角为(0,0)，该属性格式说明如下：

(1) 矩形：“x1，y1，x2，y2”，分别为矩形左上角坐标 (x1,y1) 和右下角坐标 (x2,y2)

(2) 圆形：“x，y，r”，分别为圆形坐标 (x,y) 和半径 r

(3) 多边形：“x1，y1，x2，y2，x3，y3 … ”，分别为多边形的顺序顶点坐标

<img>使用的 ismap 属性是个布尔属性，该属性将图像定义为服务器端的图像映射，当单击一个服务器端的图像映射时，单击处的坐标会以 URL 查询字符串的形式发送到服务器。

使用<audio>元素

<audio>用于定义声音，比如音乐或其他音频流。该属性目前支持3种文件格式：MP3、WAV、Ogg。可以在<audio>和</audio>之间放置文本，当浏览器不支持该元素时显示。该元素的常用属性如下：

属性 描述
autoplay 布尔属性，在加载完成后马上播放音频
controls 布尔属性 ，控制是否显示音频控件
loop 布尔属性，循环播放音频
muted 布尔属性，将音频输出为静音
preload 规定是否在页面加载后载入音频，取值有如下3个：Auto（当页面加载后载入整个音频），meta（当页面加载后只载入元数据）,none（当页面加载后不在音频）。如果设置了 autoplay 属性，则忽略 preload 属性
src 指定音频文件的 URL

<source>元素用来为媒体元素定义媒体资源，通过 src 属性指定媒体文件的 URL，通过 type 属性指定媒体资源的 MIME 类型。如果不确定浏览器支持的音频格式，则可不使用 src 属性，而使用多个 <source>元素链接不同的音频文件，浏览器将使用第一个支持的音频文件。

使用<video>元素

<video>元素提供播放、暂停和音量控件来控制视频，同时提供 width 和 height 属性来控制视频的尺寸，如果没设置这些属性，浏览器就不会在加载时保留特定的空间。在<video>标签对之间插入的内容被提供给不支持<video>元素的浏览器显示。<video>元素拥有和<audio>元素类似的方法，属性和事件。<video>元素的方法、属性和事件也可使用 JavaScript 进行控制。

使用<embed>元素

<embed>元素可以播放音频、视频和Flash动画等，该元素用法与<audio>和<video>类似，<embed>标签定义了一个容器，用来嵌入外部应用或互动程序（插件）。

在网页中嵌入腾讯视频

在腾讯视频网站任意打开一个想要嵌入的视频，在浏览器的地址栏得到播放地址：https://v.qq.com/x/cover/m441e3rjq9kwpsc/m00253deqqo.html 在这个地址中我们需要的是文件名 m00253deqqo.html ，其中 m00253deqqo 是视频的 vid。

<iframe>元素自身与视频没有任何关系，该元素也叫“内联框架”，是一种特殊的元素，可以使用 src 属性在一个网页中嵌入另一个网页，通过修改 vid 改变要嵌入的视频。
